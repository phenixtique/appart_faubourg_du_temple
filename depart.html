Checklist de départ

  1. 1
    Vaisselle

    Faites la vaisselle ou lancez le lave-vaisselle avant de partir. Merci de vider et ranger la machine propre.

    Dishes

    Wash the dishes or run the dishwasher before leaving. Please empty and put away once clean.

  2. 2
    Poubelles

    Videz les poubelles et déposez-les dans le local dédié (rez-de-chaussée, [emplacement]).

    Trash

    Empty the bins and take them to the bin room (ground floor, [location]).

  3. 3
    Linge de lit & serviettes

    [ex : Laissez le linge de lit et les serviettes [sur le lit / dans la salle de bain / dans le panier prévu]]. Pas besoin de les laver.

    Bed linen & towels

    [e.g. Leave bed linen and towels [on the bed / in the bathroom / in the basket provided]]. No need to wash them.

  4. 4
    Rangement général

    Rangez les objets utilisés à leur place. Vous n'avez pas besoin de tout aspirer/nettoyer en profondeur, juste de laisser les lieux corrects.

    Tidying up

    Put back items you've used. You don't need to deep clean, just leave the place in decent order.

  5. 5
    Éteindre & fermer

    Éteignez toutes les lumières, la climatisation/chauffage, la TV et les appareils électroniques. Fermez les fenêtres.

    Switch off & close

    Turn off all lights, AC/heating, TV, and electronics. Close all windows.

  6. 6
    Clés

    Remettez les clés dans la boîte à clés (même emplacement qu'à l'arrivée) et refermez-la avec le code.

    Keys

    Return the keys to the lockbox (same location as on arrival) and close it with the code.

Remise des clés

🗝️ Où laisser les clés

Replacez les clés dans la boîte à clés à l'entrée de l'immeuble.

Refermez-la en composant le code et vérifiez qu'elle est bien fermée.

Si vous avez perdu une clé, contactez-nous immédiatement.

Return the keys to the lockbox at the building entrance.

Close it with the code and check it is securely shut.
